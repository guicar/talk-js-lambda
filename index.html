λ-calcul

  • langage de programmation
  • années 30
  • uniquement des fonctions

Définition

  • variable : x
  • fonction : x => e
  • application : f(x)

Notation

Traditionnellement :

  • fonction : λx. e
  • application : f x

En Elm :

  • fonction : \x -> e
  • application : f x
